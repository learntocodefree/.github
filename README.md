# LearnToCodeFree Repository Guide

This repository contains the main community, contribution, and support documentation for LearnToCodeFree. Use the links below to quickly jump to the file you need.

## Quick Links

- [profile/README.md](profile/README.md) — The public organization profile, mission, and overview of what LearnToCodeFree offers.
- [CONTRIBUTING.md](CONTRIBUTING.md) — Guidelines for contributing improvements, fixes, tutorials, and open-source resources.
- [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) — Community rules and expected behavior for respectful collaboration.
- [SECURITY.md](SECURITY.md) — How to responsibly report security vulnerabilities.
- [SUPPORT.md](SUPPORT.md) — Where to ask for help, discussions, and issue guidance.
- [PULL_REQUEST_TEMPLATE.md](PULL_REQUEST_TEMPLATE.md) — A checklist template for preparing pull requests.
- [ISSUE_TEMPLATE/bug_report.md](ISSUE_TEMPLATE/bug_report.md) — Template for reporting bugs and issues.
- [ISSUE_TEMPLATE/feature_request.md](ISSUE_TEMPLATE/feature_request.md) — Template for suggesting new features or improvements.
- [ISSUE_TEMPLATE/config.yml](ISSUE_TEMPLATE/config.yml) — Settings for issue templates and external contact links.

## Start Here

- New contributor: read [CONTRIBUTING.md](CONTRIBUTING.md)
- Need help: check [SUPPORT.md](SUPPORT.md)
- Reporting a security issue: use [SECURITY.md](SECURITY.md)
- Want to share feedback or ideas: open the relevant issue template under [ISSUE_TEMPLATE](ISSUE_TEMPLATE)
