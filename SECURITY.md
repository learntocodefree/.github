# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

Do not create a public issue for security-related vulnerabilities.

Instead, contact:

[github@learntocodefree.com](mailto:github@learntocodefree.com)

Please include:

* Description of the issue
* Steps to reproduce
* Potential impact
* Suggested fix (if known)

We will review all reports and respond as quickly as possible.

Thank you for helping keep LearnToCodeFree secure.
